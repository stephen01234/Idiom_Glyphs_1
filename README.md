# Idiom_Glyphs_1
https://stephen01234.github.io/Idiom_Glyphs_1/

綜合學習卡 (Comprehensive Learning Cards)

這是一個基於 Web 的互動式學習工具，專為中文生字、成語及字義複習而設計。透過翻卡記憶、隨機抽考與即時搜尋功能，幫助使用者更有效率地掌握學習內容。專案採用單一 HTML 檔案架構，結合 React 與 Tailwind CSS，輕量且易於部署（支援 GitHub Pages）。

✨ 主要特色 (Key Features)

1. 三大核心學習模式

本系統提供三種針對不同學習目標的模式，使用者可透過頂部選單自由切換：

🔤 字卡模式 (Glyphs) - 生字認讀

功能目標：強化對生字的字形記憶與讀音連結。

操作方式：

選擇冊別：從下拉選單選擇要練習的課本冊數。

選擇練習模式：

字音練習：卡片正面顯示「生字」，背面顯示「拼音」，測試讀音記憶。

字型練習：卡片正面顯示「拼音」，背面顯示「生字」，測試字形書寫記憶。

混合模式：隨機出現上述兩種題型，增加挑戰性。

翻卡互動：點擊卡片中央即可翻面查看正確答案。

📖 成語模式 (Idioms) - 意涵理解

功能目標：學習成語的正確寫法與含義。

操作方式：

選擇冊別：篩選特定範圍的成語，適合分階段複習。

翻卡學習：正面顯示成語，點擊後背面顯示詳細解釋。

搭配功能：遇到不熟悉的成語，點擊卡片右上角的「星星」加入收藏，稍後可將冊別切換至「我的收藏」進行集中複習。

📝 字義模式 (Meanings) - 多義詞與檢索

功能目標：深入理解同一個字在不同詞語中的變化與意義。

提供三種篩選檢視模式（於「篩選模式」選單切換）：

依冊別：傳統翻卡模式，依課本進度複習詞語解釋。

依多義字詞：系統會自動偵測資料庫，列出出現 2 次以上 的字詞（例如：「白」在「雪白」與「真相大白」中的不同意思）。選擇特定字詞後，卡片僅會出現該字的相關詞語，方便比較差異。

🔍 查字搜尋 (列表)：

適合快速查詢。切換至此模式後，輸入目標關鍵字（如「心」）。

系統會以表格形式，列出所有包含該關鍵字的詞語、出處冊別及解釋。

表格左側提供星星按鈕，可直接進行收藏操作。

2. 實用學習工具

⭐ 雲端收藏 (Star/Bookmark)：整合 Firebase Firestore，所有模式皆支援收藏功能。點擊星星後，資料會同步至雲端，方便跨裝置複習不熟的內容。

🔀 隨機抽考：點擊「打亂順序」按鈕，可將目前的卡片堆順序隨機化，打破依順序記憶的慣性。

📂 彈性資料來源：

自動同步：系統預設從 GitHub Repository 讀取 glyphs.json、idioms.json 等資料檔。

CSV 上傳：支援使用者上傳自定義的 CSV 檔案，立即擴充題庫（格式請見下方說明）。

3. 響應式設計 (RWD)

採用 Tailwind CSS 打造，無論在電腦螢幕、平板或手機上操作，版面皆會自動適應，提供最佳閱讀體驗。

🚀 如何使用 (How to Use)

資料格式說明

若您希望建立自己的題庫，請準備以下格式的 JSON 檔案並放置於專案根目錄，或是透過介面上傳 CSV。

1. 字卡 (glyphs.json / CSV)

CSV 欄位順序：冊別, 生字, 拼音, 詞語

範例：

第一冊, 貓, māo, 花貓



2. 成語 (idioms.json / CSV)

CSV 欄位順序：冊別, 成語, 解釋

範例：

第一冊, 一石二鳥, 比喻做一件事獲得兩種效果。



3. 字義 (meanings.json / CSV)

CSV 欄位順序：冊別, 完整詞語, 目標字詞, 解釋

範例：

第一冊, 雪白, 白, 像雪一樣的顏色。
第一冊, 真相大白, 白, 顯露、揭開。

Backend / Persistence: Firebase (Authentication & Firestore)
